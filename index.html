🤖 Jarvis AI Voice Assistant

A modular AI-powered voice assistant built with Python. Supports voice commands, automation, memory systems and AI agents.

View GitHub Repository Download EXE

Features

🎤 Voice Commands

Control your system and interact using natural speech.

🧠 AI Agents

Modular AI agent system designed for intelligent task handling.

🖥 Jarvis HUD

Futuristic UI inspired by Iron Man's Jarvis interface.

⚡ Automation

Execute system commands and automate productivity tasks.

📚 Memory System

Store knowledge and improve assistant responses over time.

🔊 Voice Responses

Natural text-to-speech responses for smooth interaction.

Project Architecture


Voice_assistant_anti
│
├── brain/        AI logic and agents
├── commands/     Command handlers
├── core/         Audio engine & runtime
├── ui/           Jarvis visual interface
├── config/       Configuration files
│
├── main.py       Entry point
└── requirements.txt

Installation


git clone https://github.com/mishraji018/Voice_assistant_anti

cd Voice_assistant_anti

pip install -r requirements.txt

python main.py
